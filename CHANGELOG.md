# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-10

### Added

- Initial release of dockerignore Templates project
- 8 programming language templates: Node.js, Python, Java, Go, Rust,
  Ruby, PHP, TypeScript
- 13 framework templates: React, Angular, Vue, Svelte, Next.js, Django, Laravel,
  Spring Boot, ExpressJS, FastAPI, Flask, Phoenix, Rails
- 5 tool templates: Global, Docker, Git, IntelliJ, VSCode
- Comprehensive project documentation and organization

### Documentation

- Created README.md with project overview, usage instructions, and contribution guidelines
- Created AGENTS.md handover document for project maintenance
- Created LICENSE.md with MIT License
- Created CHANGELOG.md for tracking changes

### Notes

Initial release providing comprehensive `.dockerignore` templates for various
technologies to optimize Docker builds by excluding unnecessary files from
build contexts.
